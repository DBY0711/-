📡 RPA 调度 API 文档

🔑 认证说明

除 /api/yingdao/callback 外，所有写操作需在请求头携带：

X-API-Token: rpa-schedule-token-2026

可设置环境变量 API_TOKEN 自定义。

GET 系统信息

/api/tasks

获取任务列表

/api/yingdao/run-history?page=1&per_page=50&status=&search=

分页查询运行记录，支持 status 筛选和 search 搜索

/api/yingdao/callbacks

查看原始回调日志（最近100条）

/api/alerts?status=异常

查询告警记录（status=异常/成功）

/api/check-login

检查登录状态

POST 任务 & 同步

/api/tasks
{"tasks":[...],"nid":100}

保存任务数据

/api/yingdao/sync
{}

从影刀同步计划+运行记录，合并到本地

/api/yingdao/start
{"scheduleUuid":"...","robotUuid":"...","pc":"rpa5@szlc","process":"流程名"}

启动任务（自动调 schedule/detail 获取正确 robotUuid）

/api/yingdao/rerun
{"jobUuid":"...","process":"流程名"}

手动重跑失败任务

POST 登录

/api/login
{"username":"admin","password":"rpa2026!"}
/api/logout

POST 错误报告 ⭐

⚠ 需 API Token，不需要登录

接收外部失败报告，自动匹配本地任务并判断是否重试。

/api/error-report

支持三种格式：

1. JSON 直接格式：

Content-Type: application/json

{"process":"视频号-直播-数据大屏","msg":"连接超时","pc":"rpa7@szlc"}

2. JSON 嵌套格式（真实告警）：

Content-Type: application/json

{"body":{"message":"时间：2026-06-06 08:03:42；运行机器人：rpa7@szlc；运行异常计划：视频号-直播-数据大屏；错误信息：超时"}}

3. 纯文本格式：

Content-Type: text/plain

时间：2026-06-06 08:03:42；运行机器人：rpa7@szlc；运行异常计划：视频号-直播-数据大屏；错误信息：超时

文本格式支持的字段：

Key映射到
运行异常计划process（流程名）
运行异常应用process（备选）
运行机器人pc（电脑名）
错误信息msg（错误详情）
时间reportTime

返回示例：

{"success":true,"matched":true,"retried":true,"retryCount":1,"message":"已触发重试: 视频号-直播-数据大屏（第1次）"}

curl 示例：

curl -X POST http://121.40.245.6/api/error-report \
  -H "Content-Type: application/json" \
  -H "X-API-Token: rpa-schedule-token-2026" \
  -d '{"process":"视频号-直播-数据大屏","msg":"连接超时","pc":"rpa7@szlc"}'

POST 影刀回调（无需认证）

/api/yingdao/callback

影刀平台回调地址。自动验证签名、记录运行历史、按规则重试。

{"headers":{...},"params":{"sign":"...","bodyMd5":"..."},"body":{"robotName":"抖音加热-短视频","status":"error","jobUuid":"...","robotClientName":"rpa7@szlc","startTime":"...","endTime":"..."}}

PUT / DELETE 告警 & 记录

PUT /api/alerts/:id
{"checked":true,"note":"已处理"}
DELETE /api/alerts/:id
DELETE /api/yingdao/run-history/:id
DELETE /api/yingdao/run-history

清空全部运行记录

DELETE /api/yingdao/callbacks/:id
DELETE /api/yingdao/callbacks

清空全部回调日志
